講座①：パソコンと”仲良く”なるための基本のキ

はじめまして！これから、あなたと一緒に「AIと共に創る」という、ワクワクする冒険の旅に出ることを、心から楽しみにしています。 本格的な冒険の前に、まずはあなたの相棒となるパソコンと、しっかり”仲良く”なるところから始めましょう。家を建てる前に、地面をしっかり固めるのと同じです。ここで学ぶことは、一つ一つはとても簡単ですが、これからのあなたの創作活動を、驚くほどスムーズにしてくれる大切な土台となりますよ。

第1章：情報の整理整頓術 - ファイルとフォルダの作り方・片付け方

あなたのパソコンの中は、言わばあなたの「デジタルな作業机」です。机の上がゴチャゴチャだと、いざという時に必要な道具や書類が見つからなくて、困ってしまいますよね。 この章では、あなたの作業机をいつもスッキリ綺麗に保つための、魔法のように簡単な整理整頓術を学びます。これができるようになるだけで、「あのファイル、どこに保存したっけ…？」と探す時間がなくなり、創作活動にもっと集中できるようになりますからね！

まずは言葉の準備運動！「ファイル」と「フォルダ」

「ファイル」とは、写真1枚、メモ書き1つといった、情報の最小単位のこと。例えるなら、机の上の「一枚一枚の書類や写真」です。 「フォルダ」とは、それらのファイルをまとめて入れておくための「入れ物」のこと。机の上の「クリアファイル」や「引き出し」をイメージしてください。書類をクリアファイルに仕分けるように、ファイルをフォルダに片付けていくのです。

さあ、やってみよう！ 最初の「フォルダ」を作ってみる

まずは、よく使う書類をまとめておく「引き出し」にあたる、「フォルダ」を作ってみましょう。今回は、パソコンのデスクトップ（最初の画面）に作ってみますね。

【Windows の場合】
  1. デスクトップの、アイコンが何もない場所で「右クリック」します。
  2. 黒いメニューが出てくるので、その中の「新規作成(X)」にマウスカーソルを合わせます。
  3. さらに右側にメニューが出てくるので、一番上にある「フォルダー(F)」をクリックします。
  4. 「新しいフォルダー」という名前のアイコンがデスクトップに現れたら、成功です！
  5. 青く反転している状態ですぐにキーボードで好きな名前（例えば「練習用」）を打ち込み、Enterキーを押せば名前を付けられます。
【Mac の場合】
  1. デスクトップの、アイコンが何もない場所で「右クリック」します。（※トラックパッドの場合は、指2本でタップ）
  2. メニューが出てくるので、一番上にある「新規フォルダ」をクリックします。
  3. 「名称未設定フォルダ」という名前のアイコンがデスクトップに現れたら、成功です！
  4. 名前の部分が入力できる状態になっているので、そのままキーボードで好きな名前（例えば「練習用」）を打ち込み、Enterキー（returnキー）を押せば完了です。

【追記】作った後から名前を変える方法

フォルダを作った時にすぐ名前をつけ忘れてしまっても、ご安心ください！いつでも簡単に名前は変更できますよ。

＜Windows の場合＞
  1. 名前を変えたいフォルダを一度だけクリックして選びます。
  2. 選んだフォルダの上で「右クリック」します。
  3. 出てきたメニューの中から「名前の変更(M)」（四角いアイコンが目印です）をクリックします。
  4. 名前が入力できるようになるので、新しい名前をキーボードで入力し、最後にEnterキーを押せば完了です！
＜Mac の場合＞
  1. 名前を変えたいフォルダを一度だけクリックして選びます。
  2. 選んだフォルダの上で「右クリック」（または指2本でタップ）します。
  3. 出てきたメニューの中から「名前を変更」をクリックします。
  4. 名前の部分が入力できるようになるので、新しい名前をキーボードで入力し、最後にreturn（Enter）キーを押せば完了です！

さあ、最初のクエストです！

あなたの冒険の証を記録していくために、デスクトップに「冒険の記録」という名前のフォルダを作ってみましょう！ これから先のステップで創っていく画像や動画は、すべてこの「宝箱」に大切にしまっていくことにしますよ。

次は、ファイルをフォルダにお片付け！

フォルダという「引き出し」ができたので、次はファイルという「書類」をそこにお片付けしてみましょう。

  1. デスクトップ上にある、どれか一つのアイコン（ファイルや、先ほど作ったフォルダでもOK）をマウスでクリックしたまま、離さずに動かしてみてください。これが「ドラッグ」です。
  2. そのアイコンを、先ほど作った「練習用」フォルダのアイコンの上までズルズルと引きずっていきましょう。
  3. フォルダの上が少し光ったり、色が変わったりしたら、マウスのボタンを離します。 これを「ドロップ」と言います。
  4. デスクトップからアイコンが消え、「練習用」フォルダをダブルクリックして開くと、中にさっきのアイコンが入っているはずです。これがお片付けの基本です！

お疲れ様でした！

たったこれだけ？と思うかもしれませんが、この「フォルダを作って、そこにファイルを片付ける」という基本操作こそが、整理整頓術の神髄です。 「旅行の写真」「仕事の書類」のように、ご自身でルールを決めてフォルダ分けをするだけで、あなたのパソコンは驚くほど使いやすくなりますよ。

第2章：魔法の呪文「ショートカットキー」 - 作業効率を劇的に上げる方法

第1章、お疲れ様でした！ これであなたは、パソコンの中をいつでもスッキリ綺麗に保つ「整理整頓術」を身につけました。素晴らしい第一歩です！ さて、この第2章では、あなたのパソコン操作を「え、速い！まるで魔法使いみたい！」と言われるレベルに引き上げる、とっておきの呪文を伝授します。その名も「ショートカットキー」です。

「ショートカットキー」って、いったい何？

ショートカットキーとは、キーボードの特定のキーを組み合わせて押すことで、マウスを何度もカチカチしなくても、パソコンに命令を出すことができる「秘密の合言葉」のようなものです。 例えば、今までマウスで「ファイル」→「コピー」と3回クリックしていた作業が、キーボードを「タンッ！」と一度押すだけで完了してしまいます。長い道のりを歩く代わりに、秘密の近道（ショートカット）を知っているようなものですね。

まずは、最強の相棒キーを覚えよう！

ショートカットキーの世界には、主役となる2人の「王様」がいます。

  • Windowsのパソコンなら → [Ctrl] (コントロール) キー
  • Macのパソコンなら → [command] (⌘) キー

このキーは、キーボードの一番下の列、スペースキーの近くにあります。これから紹介する呪文は、ほとんどすべて、この「王様キー」を押しながら、もう一つのキーを押す、という組み合わせで発動しますよ。

これだけは覚えたい！基本の呪文ベスト５

呪文はたくさんありますが、全部を一度に覚える必要はありません。まずは、これから紹介する「本当に使える５つの基本呪文」だけをマスターしましょう。これだけで、あなたの作業時間は半分になるかもしれません！

呪文①：コピーの術【Ctrl + C】 / 【⌘ + C】

選んだ文字やファイルを、そっくりそのまま複製する魔法です。「C」は Copy（コピー）のC、と覚えると簡単ですね。

呪文②：貼り付けの術【Ctrl + V】 / 【⌘ + V】

コピーしたものを、好きな場所に貼り付ける魔法です。コピーの術とセットで使います。【C】の隣にあるキーなので、覚えやすいですよ。

呪文③：切り取りの術【Ctrl + X】 / 【⌘ + X】

選んだ文字やファイルを、場所を「移動」させる魔法です。コピーと違って、元の場所からは消えて、貼り付けた場所に引っ越します。「X」の形がハサミに似ている、とイメージしてください。

【実践】コピー＆貼り付けをやってみよう！

  1. 何かメモ帳やWordを開いて、「あいうえお」と文字を打ってみましょう。
  2. マウスで「あいうえお」の文字をドラッグして、選択状態（色が反転した状態）にします。
  3. キーボードで [Ctrl] (または[⌘]) を押したまま、[C] のキーをポンと押します。
  4. 何も起こらないように見えますが、大丈夫。パソコンはしっかり覚えています。
  5. 文字の下の行など、好きな場所を一度クリックしてから、今度は [Ctrl] (または[⌘]) を押したまま、[V] のキーをポンと押します。
  6. どうでしょう？「あいうえお」が貼り付けられたら、魔法は成功です！
呪文④：元に戻す術【Ctrl + Z】 / 【⌘ + Z】

「あ！間違えた！」という時に、操作を一つ前に戻すことができる、最高に便利な時間逆行の魔法です。何かを間違えて消してしまっても、この呪文を唱えれば、すぐ元通り。安心して、色々な操作を試すことができますね。

呪文⑤：上書き保存の術【Ctrl + S】 / 【⌘ + S】

今開いているファイルを、上書き保存する魔法です。「S」は Save（セーブ・保存）のSです。大切なレポートや作品を作っている時は、こまめにこの呪文を唱えることで、「データが消えた！」という悲劇を防ぐことができます。

お疲れ様でした！

たくさんの呪文が出てきて、少し頭が疲れたかもしれませんね。でも、ご安心ください。自転車の乗り方と同じで、一度体が覚えてしまえば、もう意識しなくても指が自然に動くようになります。 まずは一日一回、どれか一つでも呪文を使ってみることから始めましょう。その小さな一歩が、未来のあなたの大きな時間を作ってくれますよ。

第3章：インターネットの歩き方 - 安全なブラウザの使い方と検索のコツ

第2章、お疲れ様でした！「ショートカットキー」という魔法の呪文は、これからあなたの時間をたくさん生み出してくれる、一生モノのスキルになりますよ。 さて、この第3章では、いよいよ「インターネット」という、世界中の情報が集まる巨大な海を旅する方法を学びます。この海は、素晴らしい宝物がたくさん見つかる一方で、ときには危険な場所もあります。でも、ご安心ください。安全な航海の仕方さえ覚えてしまえば、何も怖いことはありませんからね！

航海の道具「ブラウザ」とは？

インターネットの海を旅するためには、「ブラウザ」という名前の「船」に乗る必要があります。 あなたが今、この講座を見ているソフト（Google Chrome, Microsoft Edge, Safariなど）も、このブラウザの一種です。ブラウザは、インターネット上にあるホームページという「島々」を訪れて、そこに書かれている文字や写真を見せてくれる、あなたのための専用船なのです。

とても大切！ 船はいつでも最新の状態に

一つだけ、とても大切な約束があります。それは、「ブラウザは、いつでも最新のバージョンにアップデートしておく」ということです。 古いブラウザを使い続けるのは、例えるなら、船体に小さな穴が空いたまま航海を続けるようなもの。悪意のある人（海賊！）が、その穴から侵入して、あなたのパソコンから大切な情報を盗んだり、パソコンをウイルスで病気にさせたりするかもしれません。 アップデートのお知らせが来たら、面倒くさがらずに、必ず実行するようにしましょう。それが、あなたの安全を守る一番の基本です。

欲しい宝物を見つける「検索のコツ」

インターネットの海は、あまりにも広大なので、やみくもに航海しても、目的の島（情報）にはたどり着けません。そこで活躍するのが、GoogleやYahoo!といった「検索エンジン」という、超優秀な「羅針盤」です。 この羅針盤を上手に使うには、少しだけコツがいります。上手な「行き先の伝え方」を覚えましょう！

テクニック 使い方 例えるなら…
単語を区切る 「渋谷 ランチ パスタ」のように、キーワードをスペースで区切って入力する。 「渋谷で、ランチに、パスタが食べたい」と、要点を伝えている感じ。
フレーズで探す "夏目漱石 吾輩は猫である" のように、「" "」(ダブルクォーテーション)で囲む。 「夏目漱石の『吾輩は猫である』という、まさにこの題名の本を探して！」と、本の名前を正確に伝えている感じ。
除外する 「ジャガー -車」のように、含めたくない単語の前に「-」(マイナス)を付ける。 「動物のジャガーについて知りたい。でも、自動車の情報は要りません」と、いらないものを伝えている感じ。

安全な航海のための３つの心得

最後に、危険な海賊から身を守るための、大切な「３つの心得」をお伝えします。

  1. サイトの「鍵マーク」を確認する
    ブラウザの上の方、アドレスが表示されている欄の左端に、南京錠のような「鍵マーク」があるか確認しましょう。このマークがあるサイトは、「通信が暗号化されていますよ」という安全の証です。特に、個人情報やクレジットカード番号を入力する際は、必ずこの鍵マークがあることを確認してくださいね。
  2. 怪しい「エサ」に食いつかない
    「おめでとうございます！当選しました！」「あなたのPCは危険です！」といった、あなたを驚かせたり、喜ばせたりするようなメールやポップアップ広告には、注意が必要です。それは、あなたの情報を盗むための「釣り針」かもしれません。知らない人からの甘い誘いや、急かすような警告は、まず疑ってみましょう。
  3. 見知らぬ「お土産」は持ち帰らない
    インターネット上には、無料でダウンロードできるソフトやファイル（お土産）がたくさんあります。しかし、中にはコンピューターウイルスという「悪魔」が潜んでいるものも…。提供元がよくわからない、怪しいサイトからのダウンロードは絶対にやめましょう。

お疲れ様でした！

これであなたも、広大なインターネットの海を安全に、そして自由に冒険するための知識を身につけました。 早速、今日学んだ検索のコツを使って、あなたの好きなこと（例えば「初心者向け 編み物 コツ」や「おいしいコーヒーの淹れ方」など）を調べてみてください。きっと、素晴らしい宝物が見つかりますよ！

第4章：【トラブル解決】困った時のための最初の応急処置

第3章、お疲れ様でした！ これであなたはもう、インターネットという広大な海を安全に冒険できる、立派な船乗りです。 いよいよ、この「ステップ0：冒険の準備編」も最後の章となりました。ここでは、あなたの相棒であるパソコンの「なんだか調子が悪い…」「動かなくなっちゃった！」という時に、あなたが自分でできる、最初の応急処置の方法をお伝えします。 お医者さんを呼ぶ前に、ちょっとした腹痛なら薬を飲んで様子を見ますよね。それと同じで、パソコンのトラブルのほとんどは、これからお教えする簡単な方法で解決してしまうのです。一番大切なのは、慌てないこと。深呼吸をして、一つずつ試していきましょう。

症状①：パソコンが完全に固まって、動かなくなった！

マウスを動かしても矢印が動かない、キーボードを叩いても反応がない… まるでパソコンが考え事をしすぎて、フリーズしてしまったような状態ですね。

  1. まずは、1分だけ待ってみる
    パソコンが、たくさんの仕事を一度にやろうとして、一生懸命になっているだけかもしれません。まずは深呼吸をして、少しだけ待ってあげましょう。意外と、これで動き出すことも多いんですよ。
  2. 特定のソフトだけを終了させる
    特定のソフト（例えば、ブラウザだけ）が原因で固まっている場合、そのソフトだけを強制的に終了させることができます。
    • Windowsの場合: [Ctrl] + [Alt] + [Delete] キーを同時に押して、「タスクマネージャー」を起動し、動いていないプログラムを選んで終了させます。
    • Macの場合: [command ⌘] + [option] + [esc] キーを同時に押して、「アプリケーションの強制終了」ウィンドウを出し、動いていないプログラムを選んで終了させます。
  3. 最終手段：「再起動」
    何をしてもダメな場合の、最強の魔法が「再起動」です。電源ボタンを5秒以上長押しして、一度パソコンの電源を完全に切り、もう一度電源を入れ直してみましょう。

症状②：インターネットに繋がらない！

昨日まで見られたホームページが見られない時。情報の海への道が、どこかで途切れてしまっているのかもしれません。

  1. Wi-Fiマークを確認する
    画面の右下（Windows）や右上（Mac）にある、扇形のWi-Fiマークを見てみましょう。そこに「！」がついていたり、いつもと違う表示になっていたりしませんか？もしそうなら、一度Wi-Fiをオフにして、もう一度オンにし直してみましょう。
  2. 他の機械は繋がるか確認する
    あなたのスマートフォンや、他のパソコンはインターネットに繋がりますか？もし他の機械も繋がらないなら、問題はパソコンではなく、お家のインターネット環境にある可能性が高いです。
  3. ルーターを再起動する
    インターネットの入口を守っている「ルーター」という機械が、少し疲れているのかもしれません。ルーターの電源コードを一度抜いて、1分ほど待ってから、もう一度差し込んでみましょう。これでお家のインターネットが、まるごと元気になることがよくあります。

困った時の最強のおまじない：「再起動」

「なんだか動きが遅いな」「いつもと様子が違うな…」
そんな、原因がよくわからない不調の時にも、「再起動」は絶大な効果を発揮します。 パソコンをずっと付けっぱなしにしていると、人間が寝不足になるように、パソコンの中にも見えない疲れが溜まっていきます。再起動は、パソコンに一度しっかりお休みを取らせて、頭をスッキリさせてあげるようなもの。トラブルの9割は、再起動で直ると言われているほど、万能な応急処置なのです。
