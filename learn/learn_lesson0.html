第1章：情報の整理整頓術 - ファイルとフォルダの作り方・片付け方

あなたのパソコンの中は、言わばあなたの「デジタルな作業机」です。机の上がゴチャゴチャだと、いざという時に必要な道具や書類が見つからなくて、困ってしまいますよね。この章では、あなたの作業机をいつもスッキリ綺麗に保つための、魔法のように簡単な整理整頓術を学びます。

まずは言葉の準備運動！「ファイル」と「フォルダ」

「ファイル」とは、写真1枚、メモ書き1つといった、情報の最小単位のこと。例えるなら、机の上の「一枚一枚の書類や写真」です。
「フォルダ」とは、それらのファイルをまとめて入れておくための「入れ物」のこと。机の上の「クリアファイル」や「引き出し」をイメージしてください。

さあ、やってみよう！ 最初の「フォルダ」を作ってみる

【Windows の場合】
  1. デスクトップの何もない場所で「右クリック」します。
  2. メニューの「新規作成(X)」にカーソルを合わせ、「フォルダー(F)」をクリックします。
  3. 「新しいフォルダー」が現れたら成功！好きな名前（例：「練習用」）を打ち込み、Enterキーを押します。
【Mac の場合】
  1. デスクトップの何もない場所で「右クリック」（または指2本でタップ）します。
  2. メニューの「新規フォルダ」をクリックします。
  3. 「名称未設定フォルダ」が現れたら成功！好きな名前を打ち込み、Enterキーを押します。

第2章：魔法の呪文「ショートカットキー」 - 作業効率を劇的に上げる方法

この第2章では、あなたのパソコン操作を「え、速い！まるで魔法使いみたい！」と言われるレベルに引き上げる、とっておきの呪文「ショートカットキー」を伝授します。

「ショートカットキー」って、いったい何？

キーボードの特定のキーを組み合わせて押すことで、マウスを何度もカチカチしなくても、パソコンに命令を出すことができる「秘密の合言葉」のようなものです。

これだけは覚えたい！基本の呪文ベスト５

やりたい事 Windowsの呪文 Macの呪文 覚え方
コピー（複製） Ctrl + C ⌘ + C Copy (コピー) のC
貼り付け Ctrl + V ⌘ + V Cの隣
切り取り（移動） Ctrl + X ⌘ + X ハサミの形
元に戻す Ctrl + Z ⌘ + Z 一番頼りになるZ
上書き保存 Ctrl + S ⌘ + S Save (セーブ) のS

第3章：インターネットの歩き方 - 安全なブラウザの使い方と検索のコツ

この第3章では、「インターネット」という、世界中の情報が集まる巨大な海を旅する方法を学びます。安全な航海の仕方さえ覚えてしまえば、何も怖いことはありません。

安全な航海のための３つの心得

  1. サイトの「鍵マーク」を確認する：アドレスバーの左端にある鍵マークは、安全の証です。
  2. 怪しい「エサ」に食いつかない：「当選しました！」といった甘い誘いや、急かすような警告は、まず疑ってみましょう。
  3. 見知らぬ「お土産」は持ち帰らない：提供元がよくわからない、怪しいサイトからのダウンロードは絶対にやめましょう。

第4章：【トラブル解決】困った時のための最初の応急処置

パソコンの「なんだか調子が悪い…」という時に、あなたが自分でできる、最初の応急処置の方法をお伝えします。一番大切なのは、慌てないことです。

困った時の最強のおまじない：「再起動」

「なんだか動きが遅いな」「いつもと様子が違うな…」
そんな、原因がよくわからない不調の時、「再起動」は絶大な効果を発揮します。パソコンに一度しっかりお休みを取らせて、頭をスッキリさせてあげるようなもの。トラブルの9割は、再起動で直ると言われているほど、万能な応急処置なのです。
