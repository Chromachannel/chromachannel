本格的な冒険の前に、まずはあなたの相棒となるパソコンと、しっかり”仲良く”なるところから始めましょう。家を建てる前に、地面をしっかり固めるのと同じです。ここで学ぶことは、一つ一つはとても簡単ですが、これからのあなたの創作活動を、驚くほどスムーズにしてくれる大切な土台となりますよ。

第1章：情報の整理整頓術 - ファイルとフォルダの作り方・片付け方

あなたのパソコンの中は、言わばあなたの「デジタルな作業机」です。机の上がゴチャゴチャだと、いざという時に必要な道具や書類が見つからなくて、困ってしまいますよね。

この章では、あなたの作業机をいつもスッキリ綺麗に保つための、魔法のように簡単な整理整頓術を学びます。これができるようになるだけで、「あのファイル、どこに保存したっけ…？」と探す時間がなくなり、創作活動にもっと集中できるようになりますからね！

まずは言葉の準備運動！「ファイル」と「フォルダ」

「ファイル」とは、写真1枚、メモ書き1つといった、情報の最小単位のこと。例えるなら、机の上の「一枚一枚の書類や写真」です。

「フォルダ」とは、それらのファイルをまとめて入れておくための「入れ物」のこと。机の上の「クリアファイル」や「引き出し」をイメージしてください。書類をクリアファイルに仕分けるように、ファイルをフォルダに片付けていくのです。

さあ、やってみよう！ 最初の「フォルダ」を作ってみる

まずは、よく使う書類をまとめておく「引き出し」にあたる、「フォルダ」を作ってみましょう。今回は、パソコンのデスクトップ（最初の画面）に作ってみますね。

【Windows の場合】
  1. デスクトップの、アイコンが何もない場所で「右クリック」します。
  2. 黒いメニューが出てくるので、その中の「新規作成(X)」にマウスカーソルを合わせます。
  3. さらに右側にメニューが出てくるので、一番上にある「フォルダー(F)」をクリックします。
  4. 「新しいフォルダー」という名前のアイコンがデスクトップに現れたら、成功です！
  5. 青く反転している状態ですぐにキーボードで好きな名前（例えば「練習用」）を打ち込み、Enterキーを押せば名前を付けられます。
【Mac の場合】
  1. デスクトップの、アイコンが何もない場所で「右クリック」します。（※トラックパッドの場合は、指2本でタップ）
  2. メニューが出てくるので、一番上にある「新規フォルダ」をクリックします。
  3. 「名称未設定フォルダ」という名前のアイコンがデスクトップに現れたら、成功です！
  4. 名前の部分が入力できる状態になっているので、そのままキーボードで好きな名前（例えば「練習用」）を打ち込み、Enterキー（returnキー）を押せば完了です。

さあ、最初のクエストです！

あなたの冒険の証を記録していくために、デスクトップに「冒険の記録」という名前のフォルダを作ってみましょう！

これから先のステップで創っていく画像や動画は、すべてこの「宝箱」に大切にしまっていくことにしますよ。

次は、ファイルをフォルダにお片付け！

フォルダという「引き出し」ができたので、次はファイルという「書類」をそこにお片付けしてみましょう。

  1. デスクトップ上にある、どれか一つのアイコン（ファイルや、先ほど作ったフォルダでもOK）をマウスでクリックしたまま、離さずに動かしてみてください。これが「ドラッグ」です。
  2. そのアイコンを、先ほど作った「練習用」フォルダのアイコンの上までズルズルと引きずっていきましょう。
  3. フォルダの上が少し光ったり、色が変わったりしたら、マウスのボタンを離します。 これを「ドロップ」と言います。
  4. デスクトップからアイコンが消え、「練習用」フォルダをダブルクリックして開くと、中にさっきのアイコンが入っているはずです。これがお片付けの基本です！

第2章：魔法の呪文「ショートカットキー」 - 作業効率を劇的に上げる方法

この第2章では、あなたのパソコン操作を「え、速い！まるで魔法使いみたい！」と言われるレベルに引き上げる、とっておきの呪文を伝授します。その名も「ショートカットキー」です。

「ショートカットキー」って、いったい何？

ショートカットキーとは、キーボードの特定のキーを組み合わせて押すことで、マウスを何度もカチカチしなくても、パソコンに命令を出すことができる「秘密の合言葉」のようなものです。

まずは、最強の相棒キーを覚えよう！

ショートカットキーの世界には、主役となる2人の「王様」がいます。

  • Windowsのパソコンなら → [Ctrl] (コントロール) キー
  • Macのパソコンなら → [command] (⌘) キー

これだけは覚えたい！基本の呪文ベスト５

呪文はたくさんありますが、全部を一度に覚える必要はありません。まずは、これから紹介する「本当に使える５つの基本呪文」だけをマスターしましょう。これだけで、あなたの作業時間は半分になるかもしれません！

呪文①：コピーの術【Ctrl + C】 / 【⌘ + C】

選んだ文字やファイルを、そっくりそのまま複製する魔法です。「C」は Copy（コピー）のC、と覚えると簡単ですね。

呪文②：貼り付けの術【Ctrl + V】 / 【⌘ + V】

コピーしたものを、好きな場所に貼り付ける魔法です。コピーの術とセットで使います。【C】の隣にあるキーなので、覚えやすいですよ。

呪文③：切り取りの術【Ctrl + X】 / 【⌘ + X】

選んだ文字やファイルを、場所を「移動」させる魔法です。コピーと違って、元の場所からは消えて、貼り付けた場所に引っ越します。「X」の形がハサミに似ている、とイメージしてください。

呪文④：元に戻す術【Ctrl + Z】 / 【⌘ + Z】

「あ！間違えた！」という時に、操作を一つ前に戻すことができる、最高に便利な時間逆行の魔法です。

呪文⑤：上書き保存の術【Ctrl + S】 / 【⌘ + S】

今開いているファイルを、上書き保存する魔法です。「S」は Save（セーブ・保存）のSです。

まとめの呪文一覧表

忘れてしまったときは、この表をいつでも見に来てくださいね。

やりたい事 Windowsの呪文 Macの呪文 覚え方

第3章：インターネットの歩き方 - 安全なブラウザの使い方と検索のコツ

この第3章では、いよいよ「インターネット」という、世界中の情報が集まる巨大な海を旅する方法を学びます。安全な航海の仕方さえ覚えてしまえば、何も怖いことはありませんからね！

航海の道具「ブラウザ」とは？

インターネットの海を旅するためには、「ブラウザ」という名前の「船」に乗る必要があります。あなたが今、この講座を見ているソフト（Google Chrome, Microsoft Edge, Safariなど）も、このブラウザの一種です。

欲しい宝物を見つける「検索のコツ」

欲しい情報を見つけるための、上手な「行き先の伝え方」を覚えましょう！

テクニック 使い方 例えるなら…
単語を区切る 「渋谷 ランチ パスタ」のように、キーワードをスペースで区切って入力する。 「渋谷で、ランチに、パスタが食べたい」と、要点を伝えている感じ。
フレーズで探す "夏目漱石 吾輩は猫である" のように、「" "」で囲む。 「まさにこの題名の本を探して！」と、正確に伝えている感じ。
除外する 「ジャガー -車」のように、含めたくない単語の前に「-」を付ける。 「動物のジャガーについて知りたい。自動車の情報は要りません」と伝えている感じ。

第4章：【トラブル解決】困った時のための最初の応急処置

パソコンの「なんだか調子が悪い…」「動かなくなっちゃった！」という時に、あなたが自分でできる、最初の応急処置の方法をお伝えします。一番大切なのは、慌てないこと。深呼吸をして、一つずつ試していきましょう。

困った時の最強のおまじない：「再起動」

「なんだか動きが遅いな」「いつもと様子が違うな…」
そんな、原因がよくわからない不調の時にも、「再起動」は絶大な効果を発揮します。トラブルの9割は、再起動で直ると言われているほど、万能な応急処置なのです。
