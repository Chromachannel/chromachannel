第1章：GIMPとは何か？ - Photopeaとの使い分けとインストールの手順

Photopeaでの仕上げ術、お見事でした！あなたはもう、AIが生み出した作品に、自らの手で魂を吹き込む術を知っています。

しかし、中にはこう感じた方もいるかもしれません。「AIで生成した、このキャラクターを、別のAIが生成した、この美しい背景と、自然に合成できないだろうか？」と。

その、尽きることのない探究心に応えるための、最強の「プロの工房」が、この講座で学ぶ「GIMP（ギンプ）」です。

Photopea工房と、GIMP工房の決定的な違い

Photopeaが「AIイラストの原石を磨き上げる、最高の研磨工房」だとしたら、GIMPは「複数の宝石を組み合わせて、全く新しい、世界で一つのジュエリーを創り出す、超本格的な彫金工房」です。

GIMPは、あなたのパソコンにインストールして使う、完全無料のソフトウェア。ブラウザで手軽に使えるPhotopeaよりも、少しだけ準備が必要ですが、その分、あなたのパソコンの性能を最大限に引き出し、より複雑で、より高度な作業を、サクサクと快適に行うことができます。

GIMP工房を、あなたのパソコンに建てる手順

GIMPは、あなたのパソコンの中に、自分だけの本格的な工房を建てるようなものです。少しだけ準備が必要ですが、一度建ててしまえば、いつでも最高の環境で創作活動ができますよ。一つ一つの工程を、一緒に、ゆっくり確認しながら進めていきましょう。

  1. ステップ①：本物の「公式サイト」という土地を探す
    まず、GIMPの公式サイトにアクセスします。インターネットの海には、公式サイトにそっくりな偽物のサイトも存在しますので、必ずこのリンクから飛ぶか、「gimp.org」という住所（URL）を確認してくださいね。本物の土地を手に入れることが、安全な工房作りの第一歩です。
  2. ステップ②：工房の「設計図」をダウンロードする
    公式サイトにアクセスすると、GIMPのマスコットキャラクター、ウィルバー君が出迎えてくれます。その近くにある、オレンジ色（または赤色）の大きな「DOWNLOAD」ボタンをクリックしましょう。これが、あなたのパソコンに工房を建てるための「設計図（インストーラー）」を手に入れるためのボタンです。
  3. ステップ③：あなたのパソコンに合った「資材」を選ぶ
    DOWNLOADボタンを押すと、いくつかの選択肢が表示されます。GIMPは、Windows、macOS、Linuxなど、色々な種類のパソコンに対応してくれているのです。
    • お使いのパソコンがWindowsなら、「Download GIMP directly」と書かれたオレンジ色のボタンが、最も簡単でおすすめです。
    • Macをお使いなら、「macOS」の項目にある、同様の「directly」と書かれたリンクをクリックしましょう。
    クリックすると、設計図のダウンロードが始まります。ファイルのサイズが少し大きいので、完了まで数分かかることもあります。美味しいコーヒーでも淹れて、リラックスしてお待ちくださいね。
  4. ステップ④：設計図と「対話」して、工房を組み立てる
    ダウンロードが完了したら、そのファイル（gimp-2.xx.xx-setup.exe のような名前です）をダブルクリックして、いよいよ組み立て作業の開始です！

    ここからは、インストーラーという名の「棟梁」が、あなたにいくつか質問をしてきますが、全く難しくありません。

    • 「このアプリがデバイスに変更を加えることを許可しますか？」→「はい」：棟梁が、あなたのパソコンという土地に、工房を建てる許可を求めています。
    • 「Select Install Mode」→「Install for all users (recommended)」：このパソコンを使う人みんなが使えるようにしますか？と聞かれています。通常はこちらでOKです。
    • 「言語の選択」→「日本語」：「OK」を押します。
    • 「GIMP セットアップウィザードへようこそ」→「インストール」：いよいよ、本格的な組み立てが始まります。あとは、棟梁が頑張ってくれるのを、応援しながら待つだけです。
  5. ステップ⑤：祝・GIMP工房、完成！
    「GIMP セットアップウィザードは完了しました。」というメッセージが表示されたら、「完了」ボタンを押します。これで、あなたのパソコンのデスクトップやスタートメニューに、GIMPのアイコンが追加されたはずです。

おめでとうございます！あなたのパソコンの中に、世界中のプロクリエイターが愛用する、最強の工房が無事に完成しました！さあ、工房の扉を開けて、新しい創造の世界へ旅立ちましょう！

第2章：プロの現場で使う、レイヤー機能と「レイヤーマスク」の基本

GIMP工房の心臓部も、Photopeaと同じく「レイヤーパネル」です。しかし、GIMPでは、このレイヤーをさらに自在に操るための、最強の魔法が使えます。それが、「レイヤーマスク」です。

この概念を理解できれば、あなたはもう、画像加工の初心者ではありません。プロの仲間入りです。世界一わかりやすく解説しますので、じっくりと、ついてきてくださいね。

【世界一わかる】「レイヤーマスク」とは、“半分だけ透明になれる魔法のすりガラス”である

「レイヤー」が透明なフィルムだったのを、覚えていますか？
「レイヤーマスク」とは、そのフィルムの上に貼り付ける、“半分だけ透明になれる、特殊なすりガラス”をイメージしてください。

このすりガラスは、とても不思議な性質を持っています。

  • このすりガラスに「黒い絵の具」を塗った部分は、完全に曇って、下のフィルムが見えなくなります（非表示）。
  • このすりガラスに「白い絵の具」を塗った部分は、完全に透明になって、下のフィルムがくっきり見えます（表示）。
  • そして、一番の魔法がこれです。このすりガラスに「灰色の絵の具」を塗った部分は、塗った灰色の濃さに応じて、半透明になり、下のフィルムがうっすらと透けて見えます。

「消しゴムツール」が、フィルムに描かれた絵そのものを“削ってしまう”のに対し、レイヤーマスクは、元の絵は一切傷つけずに、「どこを見せて、どこを隠すか」だけを、後から何度でもやり直せる形でコントロールできるのです。これが、プロが愛用する、究極の「非破壊編集」の真髄です。

第3章：実践！複数のAI画像を違和感なく合成するテクニック

さあ、レイヤーマスクという最強の魔法を使って、奇跡の合成写真を作り上げましょう！

【お題】AIで生成した「猫のキャラクター」と、別のAIが生成した「幻想的な森の背景」を、まるで最初から一枚の絵だったかのように、自然に合成する。

ステップ①：素材を工房に運び込む

  1. GIMPを起動し、「ファイル」→「開く/インポート」で、まず「幻想的な森の背景」の画像を読み込みます。これが、一番下の土台となるレイヤーです。
  2. 次に、「ファイル」→「レイヤーとして開く」で、「猫のキャラクター」の画像を読み込みます。すると、森のレイヤーの上に、新しい猫のレイヤーが重なった状態で表示されるはずです。

ステップ②：キャラクターに「魔法のすりガラス」を装着する

  1. 右下のレイヤーパネルで、「猫のキャラクター」のレイヤーが選ばれていることを確認します。
  2. レイヤーパネルの下にある、白い四角に黒い丸が描かれた「レイヤーマスクを追加」ボタンを押します。
  3. 出てきたウィンドウで「完全白（不透明）」を選んで「追加」を押すと、猫のレイヤーの隣に、真っ白な四角（レイヤーマスク）が現れます。今はまだ、何も隠していない状態ですね。

ステップ③：黒い絵の具で、不要な部分を隠していく

ここからが、レイヤーマスクの真骨頂です。

  1. 左のツールバーから「ブラシツール」を選び、描画色を「黒」にします。
  2. 【超重要】レイヤーパネルで、必ず、先ほど追加した「真っ白なレイヤーマスク」の方をクリックして、選択状態（白い枠で囲まれた状態）にしてください。
  3. この状態で、画像の「猫の周りの不要な背景」を、黒いブラシで塗りつぶしていきます。どうでしょう？ブラシで塗った部分だけが、まるで消しゴムで消したかのように透明になり、下の森の背景が見えてきたはずです！でも、元の猫の絵は一切傷ついていません。
  4. もし、間違えて猫の本体まで消してしまったら？慌てずに、ブラシの色を「白」に変えて、その部分を塗り直してみてください。すると、消えたはずの猫が、元通りに復活するはずです。これが、レイヤーマスクのすごいところ！

ステップ④：灰色の絵の具で、境界線をなじませる

キャラクターの切り抜きが終わったら、最後の仕上げです。このままでは、キャラクターと背景が、まるでシールを貼ったように、くっきりと分かれて見えてしまいます。

  1. ブラシの色を、白と黒の中間である「灰色」にします。
  2. ブラシの「硬さ」を柔らかくし、「不透明度」を下げて、半透明のブラシを作ります。
  3. この半透明の灰色ブラシで、キャラクターの輪郭、特に地面と接している足元などを、そっとなぞってあげましょう。
  4. すると、キャラクターの輪郭が、ほんの少しだけ透けて、背景の森の色と自然に溶け込んでいきます。この一手間が、作品に圧倒的な「なじみ感」と「空気感」を生み出す、プロの技なのです。

これで、まるで最初から、猫がその森にいたかのような、奇跡の一枚が完成しました！

第4章：【重要】多機能ゆえの挫折を防ぐ学習法と、PCへの負荷

4-1. GIMPという「魔境」に迷わないために

GIMPは、あまりにも多機能なため、全ての機能を一度に使いこなそうとすると、情報量の多さに圧倒され、挫折してしまいます。大切なのは、「今の自分に必要な機能だけを、一つずつ覚えていく」という姿勢です。

今回の講座で覚えるべきは、ただ一つ、「レイヤーマスクを使って、画像を自然に合成する方法」だけ。まずは、この技術を完璧にマスターすることに集中しましょう。それができれば、あなたはもう、GIMPという工房の、立派な一人前の職人です。

4-2. パソコンへの「思いやり」を忘れずに

GIMPは、Photopeaと違い、あなたのパソコンの力（メモリやCPU）を直接使って動作します。特に、たくさんのレイヤーを重ねたり、非常に大きな画像を扱ったりすると、パソコンが一生懸命になりすぎて、動きが遅くなってしまうこともあります。

作業中は、他の不要なソフトは閉じておく。こまめに「ファイル」→「保存」で作業内容を保存する。そんな、あなたの相棒であるパソコンへの、ちょっとした「思いやり」が、快適な創作活動に繋がりますよ。
