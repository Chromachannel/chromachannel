第1章：YMM4の基本操作 - タイムラインと、素材の配置

VOICEVOXでの監督業、本当にお疲れ様でした！あなたのキャラクターは今、あなたの言葉で、感情豊かに語り始めています。

さあ、いよいよステップ3のクライマックス。あなたがこれまで、丹精込めて創り上げてきた、イラスト、デザイン、台本、音楽、そして声…その全ての「素材」を、一つのキャンバスに配置し、「動画」という、究極の物語へと昇華させる時が来ました。

この、創造の最終工程を担うのが、無料でありながら、プロの現場でも通用する、奇跡のような動画編集ソフト「ゆっくりムービーメーカー4（YMM4）」です。 ゆっくりMovieMaker4（通称YMM4）は、キャラクターが会話する形式の動画、いわゆる「解説動画」や「実況動画」を作ることに特化した、驚くほど高機能で、そして直感的に使える無料の動画編集ソフトです。

このソフトの最大の特徴は、先ほど紹介したVOICEVOXと完璧に連携できる点にあります。VOICEVOXで作った音声ファイルを読み込ませるだけで、この講座の核心である「目パチ・口パク」を、ほぼ自動で実現できてしまうという、まさに魔法のような機能を秘めているのです。あなたが苦労してアニメーションを作る必要は一切ありません。

【超重要】YMM4のライセンスと2つのバージョンの違い

YMM4には、手軽な「Lite版」と、全ての機能が使える「無印版（フル機能版）」の2種類があります。ソフト自体はどちらも無料ですが、商用利用（YouTubeの収益化など）を考えている方は、ライセンスについて非常に重要な違いがあるため、必ず理解しておきましょう。

結論から言うと、この講座でVOICEVOXを使って動画を作る限り、どちらのバージョンでも問題なく、安心して商用利用まで可能です。もし迷ったら、インストールが簡単な「Lite版」から始めることを強くお勧めします！

厨房の心臓部「タイムライン」を理解する

次に、動画の舞台となるYMM4を準備します。前のセクションでお話しした通り、この講座では手軽で安心な「Lite版」を使って進めていきますね。

  1. 公式サイトにアクセスする
    こちらも、必ず公式サイトからダウンロードしましょう。
    → ゆっくりMovieMaker4公式サイト

    リンク先が起動しないときは下記をコピーして、検索窓にペースト、表示された公式サイトよりダウンロードしてください。

    コピー → ゆっくりMovieMaker4

  2. Lite版をダウンロードする
    サイトの中ほどに、「YMM4 Lite」という項目があります。そこにあるダウンロードボタンをクリックしてください。
  3. ダウンロードしたファイルを確認する
    ダウンロードが完了すると、「YMM4Lite_（日付などの数字）.zip」という名前のファイルが保存されます。この「.zip」というファイルは、複数のファイルが一つにまとまった“圧縮フォルダ”です。これを使うには、「展開（解凍）」という作業が必要になります。

    4. 【ポイント】「展開（解凍）」って、どうやるの？

    難しい言葉に聞こえますが、操作はとても簡単です。ダウンロードした「.zip」ファイルを右クリックし、メニューの中から「すべて展開」を選んでください。すると、どこに展開するかを聞かれますので、そのまま「展開」ボタンを押せばOKです。同じ場所に、通常の黄色いフォルダが新しく作成されれば、展開は成功です。

  4. フォルダの中から、YMM4を見つける
    展開してできた新しいフォルダを開いてみましょう。たくさんのファイルが入っていますが、その中から「YukkuriMovieMaker.exe」という名前のファイルを探してください。黒丸に白抜きの“ゆ”が表示されたアイコンが目印です。これが、YMM4の本体です。VOICEVOXと違い、インストール作業は必要ありません。
  5. 起動して、ショートカットを作成する
    「YukkuriMovieMaker.exe」をダブルクリックして、ソフトが起動することを確認しましょう。毎回このフォルダから探すのは大変なので、アイコンを右クリックして「ショートカットの作成」を選び、できたショートカットをデスクトップに移動させておくと、次回から便利ですよ。

タイムラインは、「物語が流れる、一本の長い道」だと考えてください。左端が「0秒地点（物語の始まり）」で、右に行けば行くほど、時間が進んでいきます。あなたの仕事は、この道の上に、「いつ」「誰が（どの素材が）」登場し、「いつ」退場するのかを、配置していくことなのです。

最初の料理：素材をタイムラインに並べてみよう

  1. YMM4の画面の、タイムライン以外の場所に、あなたが用意した素材（キャラクターのイラスト、背景画像、VOICEVOXで作った音声ファイルなど）を、ドラッグ＆ドロップで放り込んでみましょう。
  2. 放り込まれた素材は、画面右側の「素材アイテム」という棚に、きちんと整理整頓されます。
  3. さあ、いよいよ盛り付けです。その棚から、まずは「背景画像」を掴んで、タイムラインの一番下の段（レイヤー1）の、0秒地点に置いてみてください。
  4. 次に、「キャラクターのイラスト」を掴んで、背景画像の上、つまりタイムラインの2段目（レイヤー2）に置いてみましょう。
  5. 最後に、「キャラクターの音声」を掴んで、さらにその上、3段目（レイヤー3）に置きます。

画面中央の大きな「プレビュー画面」を見てください！あなたのキャラクターが、美しい背景の前に立ち、そして、あなたが作った音声に合わせて、口をパクパクさせているはずです！これが、動画編集の、最も基本的で、最も感動的な第一歩です。

第2章：視聴者を飽きさせない「カット編集」と「テロップ」の基本

素材を並べるだけなら、誰でもできます。あなたが目指すのは、視聴者を一瞬たりとも飽きさせない、テンポの良い動画を作る「名料理人」。そのための、２つの基本技術を学びましょう。

技術①：カット編集 - 会話の「えーっと…」を、切り捨てる勇気

VOICEVOXで作った音声には、言葉と言葉の間に、微妙な「間」があります。この、不要な間を切り詰めて、会話のテンポを良くする作業が「カット編集」です。

タイムライン上の音声アイテムを選択し、波形が途切れている「無音部分」に再生ヘッド（赤い縦線）を合わせ、ハサミのマークの「分割」ボタンを押す。そして、分割された不要な部分を「Delete」キーで削除し、後ろのアイテムを前に詰める。この地道な作業が、あなたの動画に、プロのような小気味よいリズムを生み出します。

技術②：テロップ - 心の声を「見える化」する魔法

キャラクターが話している内容を、文字として画面に表示するのが「テロップ」です。YMM4では、タイムライン上で右クリックし、「テキストアイテムを追加」を選ぶだけで、簡単にテロップを追加できます。

良いテロップの秘訣は、ステップ2で学んだ、Canvaのデザイン4大原則、特に「対比（コントラスト）」を意識すること。背景の色と文字の色が似ていると、視聴者は「読みにくい！」と感じ、その瞬間に動画を閉じてしまいます。文字に「フチ取り」を付けたり、半透明の「座布団」を敷いたりして、どんな背景でも、誰でも、ストレスなく読める「おもてなしの心」を忘れないでください。

第3章：動画の雰囲気を決定づける「BGM」と「効果音」の挿入テクニック

最高の料理には、最高の「音楽」と「スパイス」が欠かせません。動画におけるそれが、「BGM」と「効果音（SE）」です。

BGM - 視聴者の「感情」を支配する、見えない指揮者

BGMは、ただの飾りではありません。それは、視聴者の感情を、あなたの意図通りに誘導するための、最も強力な魔法です。楽しいシーンには、明るくアップテンポなBGMを。感動的なシーンには、静かで壮大なオーケストラを。BGMをタイムラインの一番下に、小さな音量で流し続けるだけで、あなたの動画は、一気に「作品」としての深みを増します。

効果音（SE） - 物語に「命」を吹き込む、魔法のスパイス

キャラクターが驚いた時の「キラリーン！」、面白いことを言った時の「ドゥクシ！」、重要なポイントを強調する時の「ドン！」。こうした短い「効果音」は、漫画の描き文字のように、視聴者の注意を惹きつけ、物語にリズムと生命感を与えます。

効果音は、セリフと同じレイヤーに置くのではなく、専用のレイヤー（例えば、一番上の段）に置くと、管理がしやすくなりますよ。

第4章：【最重要】著作権（BGM・効果音・画像素材）と、プロジェクト管理

あなたの厨房は今、最高の食材（あなたのオリジナル作品）と、最高の調味料（BGMや効果音）で、活気に満ち溢れています。しかし、三ツ星レストランのシェフであるためには、料理の腕前と同じくらい、大切な「心構え」があります。それは、食材の「仕入れ先」に、最大限の敬意を払うことです。

心構え①：著作権 - 他人のお店の「秘伝のタレ」を、無断で使ってはいけない

インターネット上には、「フリーBGM」「フリー効果音」といった言葉で、たくさんの素材が配布されています。しかし、この「フリー」という言葉を、「何をしても自由（Free）」だと勘違いしてはいけません。多くの場合、それは「無料（Free）」という意味であり、そこには、素材を作ってくれたクリエイターさんの想いが込められた、「利用規約」という名の、大切な“取扱説明書”が、必ず存在します。

あなたがレストランのシェフだとして、他のレストランの厨房に忍び込み、「これはフリー（無料）だから」と言って、秘伝のタレや、丹精込めて育てられた野菜を、無断で自分の料理に使ったら、どうなるでしょう？それは、窃盗であり、そのお店への、最大級の侮辱行為ですよね。

BGMや効果音、そしてあなたが使うかもしれない背景画像やイラスト素材。その一つ一つに、それを作ったクリエイターの人生と、魂が込められています。その魂への敬意が、「利用規約を読む」という、たった数分で終わる、しかし何よりも尊い行動なのです。

【実践】取扱説明書（利用規約）の、どこを読めばいいの？

「DOVA-SYNDROME」や「OtoLogic」、「効果音ラボ」といった、多くのクリエイターが愛用する、素晴らしい無料素材サイトがあります。こうしたサイトで素材をダウンロードする時、必ず、以下の「３つの項目」が書かれているページを探し、目を通す癖をつけましょう。

  1. 商用利用は可能か？：あなたのYouTubeチャンネルが、将来、収益化する可能性があるなら、これは絶対に確認しなければなりません。「商用利用OK」「営利目的OK」と明記されている素材だけを、ありがたく使わせていただきましょう。
  2. クレジット表記は必要か？：「クレジット表記不要」と書かれていれば、動画内に作者の名前などを記載する必要はありません。しかし、「クレジット表記必須」と書かれている場合、動画の最後や概要欄に、「音楽：〇〇（サイト名や作者名）」と、感謝の気持ちを込めて、必ず記載しなければなりません。これは、人としての、最低限のマナーです。
  3. 加工・改変は許可されているか？：曲の長さを変えたり、音量を調整したりする「加工」は、ほとんどの場合許可されています。しかし、元の曲が分からないほど、ズタズタに切り刻むような「改変」は、禁止されている場合もあります。

この３つの約束を守ること。それが、あなたを、ただの動画投稿者から、他のクリエイターへのリスペクトを忘れない、真の「クリエイター」へと成長させてくれるのです。

心構え②：プロジェクト管理 - 厨房を、常に整理整頓しておく

素晴らしい動画は、整理整頓された厨房から生まれます。YMM4での編集作業は、たくさんの素材ファイルを扱います。これらのファイルを、デスクトップなどにバラバラに置いておくと、後で必ず「あのファイル、どこに行った！？」という、悲劇が起こります。

動画制作を始める前に、必ず、一つの「専用フォルダ」を作り、その中に「画像」「音声」「BGM」といった、素材の種類ごとのサブフォルダを作って、きちんと仕分けしておくこと。この、ほんの少しの手間が、未来のあなたを、絶望の淵から救うことになるのです。

第5章：あなたの動画を、プロのレベルに引き上げる編集思考法

YMM4厨房での冒険、本当にお疲れ様でした！あなたはもう、動画編集の基本的な技術を、すべてその手に収めました。

しかし、最後に。技術よりも、もっと大切なことを、お伝えさせてください。プロの編集者と、アマチュアの編集者を分ける、たった一つの「思考法」の違い。それは、「視聴者の“時間”という、世界で最も貴重なものを、預かっている」という意識です。

なぜ、ここでカットするのか？それは、視聴者の集中力を、一瞬たりとも途切れさせないため。なぜ、このテロップを出すのか？それは、視聴者の理解を、一瞬でも早く助けるため。なぜ、この効果音を入れるのか？それは、視聴者の感情を、一瞬で盛り上げるため。

あなたの編集の一つ一つの判断が、視聴者への「最高のおもてなし」に繋がっているか。その問いを、常に自分に投げかけること。その、深く、そして温かい「思いやりの心」こそが、あなたの動画を、ただの映像から、人の心を動かす、忘れられない「体験」へと、昇華させるのです。

あなたの物語は、完成しましたか？

全ての素材を一つの作品へと昇華させる、動画編集の冒険、お疲れ様でした。
あなたはもう、イラスト、声、音楽、そして物語を紡ぐ、総合クリエイターです。
