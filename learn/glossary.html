用語集

AIとWebの冒険に役立つ用語集

学習の途中で「あれ、この言葉なんだっけ？」と迷子になった時、いつでもここに立ち寄ってください。
難しい専門用語を、世界一わかりやすい「例え話」で解説します。

あ行 か行 さ行 た行 は行 ま行 や行 ら行 英数字

あ行

AI（人工知能）

【例え話：見習いシェフ】人間みたいに、たくさんの情報から物事を考えたり、学んだりすることができる、賢いコンピュータプログラムのこと。たくさんのレシピ（データ）を読んで、新しい料理（答え）を考えるのが得意だよ。
関連：講座① AI教養入門

AI博士

【例え話：サイトの優しい案内役】このサイトの案内役をしてくれる、優しくて物知りなAIチャットボットのこと。あなたのどんな質問や悩みにも、共感を持って寄り添ってくれる、心強い味方だよ。
関連：AI博士の談話室

か行

機械学習 (Machine Learning)

【例え話：AIへの"教育"】AIにたくさんの「お手本（データ）」を見せてあげて、AI自身にルールやパターンを学ばせること。「仕分けマスター・ピクト」で、きみがAIに「これはくだものだよ」って教えてあげるのが、まさにこの機械学習の体験だよ。
関連：AI体験ゲーム「仕分けマスター・ピクト」

教師あり学習 / 教師なし学習

【例え話：「答えを教える育て方」と「自分で考えさせる育て方」】機械学習の、二つの異なる育て方だよ。「AIの赤ちゃん 育成アプリ」で体験したように、先生（あなた）が「これはりんごだよ」と正解を教えてあげるのが教師あり学習。反対に、AIに答えを教えず、見た目の特徴だけで自分なりに仲間分けさせるのが教師なし学習なんだ。
関連：AIの赤ちゃん 育成アプリ

画像生成AI

【例え話：魔法の絵筆】あなたが言葉で伝えたイメージ（プロンプト）を、世界に一枚だけの「絵」に変えてくれる魔法のことだよ。
関連：講座③ 画像生成AIの世界へようこそ

クリエイター

【例え話：冒険家】イラスト、音楽、動画、文章といった、新しい価値を「創造（Create）」する人のこと。このサイトは、AIという最高の相棒と一緒に、あなたを最高のクリエイターにするための学校なんだよ。

さ行

商用利用 (Commercial Use)

【例え話：作品でお金をもらうこと】あなたが作った作品（スタンプなど）を販売して、「お金（収益）」を得ることだよ。これをするためには、使ったAIツールや素材のライセンスが「商用利用OK」になっているかを、必ず確認する必要があるんだ。
関連：講座⑥ AIでLINEスタンプ制作入門

就労支援B型事業所

【例え話：自分のペースで働ける場所】障害や病気のある方が、自分のペースで働きながら、社会参加を目指すための福祉施設の一つ。このサイトの制作者も、現在利用中だよ。
関連：福祉の現場で見つけた、AIという名の「魔法の杖」

た行

著作権 (Copyright)

【例え話：作品を守るための権利】イラストや音楽、文章といった、人が創造した「作品」を守るための権利のことだよ。他人の作品を使う時は、必ずその作品の「利用規約」というルールを守って、作者への感謝と敬意を忘れないことが、クリエイターとしての一番大事な心構えなんだ。
関連：講座③の第4章：クリエイターの心構え

ディープラーニング (深層学習)

【例え話：AIの"ひらめき"】AIが人間の脳をマネして、物事の深い特徴や「概念」そのものを自分で見つけ出せるようになった、すごい勉強方法のこと。この「ひらめき」を手に入れたことで、AIは絵を描いたり、作曲したりできるようになったんだ。
関連：講座① AI教養入門

は行

ハルシネーション

【例え話：AIの"知ったかぶり"】AIが、もっともらしく、堂々と”ウソ”をついちゃうこと。AIの答えは鵜呑みにせず、必ず「裏付けを取る（ファクトチェック）」のが、AIと賢く付き合うコツだよ。
関連：講座②の第6章：AIと賢く付き合うために

背景透過

【例え話：背景を透明マントで隠す】イラストの、キャラクター以外の「背景」部分を、文字通り“透明”にすることだよ。これをすることで、スタンプがトーク画面で綺麗に表示されるんだ。Photopeaという無料ツールで、簡単にできる魔法だよ。
関連：講座⑧ Photopea入門

ベクター画像 / ラスター画像

【例え話：「切り絵」と「砂絵」】画像の作り方の違いだよ。AIが作るイラストや写真は、色のついた砂粒を集めて描く「砂絵（ラスター画像）」だから、拡大すると荒れちゃう。一方で、ロゴやアイコンは、点と線の設計図で描く「切り絵（ベクター画像）」だから、どこまでも拡大しても綺麗なままなんだ。
関連：講座⑩ Inkscape入門

ポートフォリオ

【例え話：冒険の記録】クリエイターにとっての「作品集」であり、「これまでの冒険の記録」のことだよ。あなたが「何ができるのか」を、100の言葉より雄弁に伝えてくれる、最強の武器なんだ。
関連：講座⑳ AIと創る最強ポートフォリオ

プロンプト

【例え話：AIへの"魔法の呪文"】AIに「こんなことをしてほしい！」とお願いするための「指示文」のこと。良い呪文を唱えられると、AIはもっとすごい力を発揮してくれるんだ。
関連：プロンプト集

ま行

モデル (学習モデル / チェックポイント)

【例え話：AI画家の"人格"】AIという画家の「人格」そのもののことだよ。日本のアニメばかりを学習したAIは「アニメ風モデル」に、写真ばかりを学習したAIは「写真風モデル」になるんだ。どの人格のAIにお願いするかで、絵のスタイルがガラッと変わるんだよ。
関連：講座⑤ Stable Diffusion入門

や行

UI/UXデザイン

【例え話：レストランの「内装」と「居心地」】サイトの「見た目（UI）」と「使いやすさ（UX）」をデザインするお仕事だよ。UIがレストランの「内装や食器のデザイン」なら、UXは「お店で過ごした時間全体の心地よさ」を設計するイメージだね。

ら行

リジェクト

【例え話：審査での"差し戻し"】LINEに申請したスタンプが、「ルールに合っていませんよ」と審査で“却下”されてしまうことだよ。でも、これは失敗じゃなくて、プロになるための大切な学びの機会なんだ。
関連：AIでLINEスタンプを作りたいあなたへ

レイヤーマスク

【例え話：半分だけ透明になれる魔法のすりガラス】GIMPなどで使う、プロの画像合成技術だよ。元の絵は一切傷つけずに、「どこを見せて、どこを隠すか」だけを、後から何度でもやり直せる形でコントロールできるんだ。
関連：講座⑨ GIMP入門

ロードマップ

【例え話：冒険の地図】目標を達成するまでの道のりを分かりやすく示した計画書のこと。このサイトの学習講座全体が、AIクリエイターになるための大きなロードマップになっているよ。
関連：AI学習 全講座まとめ

英数字

Canva (キャンバ)

【例え話：夢のデザインキッチン】誰でもプロみたいなデザインが作れちゃう、無料のデザインツールだよ。
関連：講座⑦ Canva入門

CSS (シーエスエス)

【例え話：ウェブサイトの"インテリア"】HTMLで作った骨格に、「デザイン」というお化粧をしてあげるための言葉だよ。壁紙の色を変えたり、家具を並べたりするイメージだね。「リアルタイムプレビューエディタ」の右側で書くのが、このCSSなんだ。
関連：HTML/CSS リアルタイムプレビューエディタ

GIMP (ギンプ) / Photopea (フォトピー)

【例え話：AIイラストの"仕上げ工房"】AIが作ったイラストの「あと一歩」を仕上げるための、プロ仕様の画像編集ツールだよ。キャラクターの指が6本になっちゃったのを直したり、複数の画像を自然に合成したりできる、最高の「彫金工房」なんだ。
関連：講座⑨ GIMP入門, 講座⑧ Photopea入門

HTML (エイチティーエムエル)

【例え話：ウェブサイトの"骨格"】ウェブページの「骨格」を作るための言葉だよ。お家を建てる時の、柱や壁みたいなものだね。「リアルタイムプレビューエディタ」の左側で書くのが、このHTMLなんだ。
関連：HTML/CSS リアルタイムプレビューエディタ

ImageFX (イメージエフエックス)

【例え話：魔法のスケッチブック】Googleが作った、誰でも簡単に使える画像生成AIだよ。「夕焼けの海辺を歩く、宇宙服を着た猫」みたいに、言葉で伝えるだけで、AIがサラサラと絵を描いてくれるんだ。
関連：講座④ ImageFX入門

JavaScript (ジャバスクリプト)

【例え話：ウェブサイトの"魔法の仕掛け"】ウェブページに「動き」や「機能」という命を吹き込むための、プログラミング言語だよ。ボタンを押したら計算してくれる家計簿や、絵文字が消えるパズルゲームなど、このサイトのほとんどのアプリが、このJavaScriptの魔法で動いているんだ。
関連：アプリ・ツール まとめ

LoRA (ローラ)

【例え話：魔法のスパイス】AI画家に、特定の画風や服装を一時的に教えるための、「追加の参考書」みたいなものだよ。「このキャラクターに、”ゴシックロリータのドレス”という参考書を渡して、着せ替えてみよう」といった、細やかなカスタマイズが可能になるんだ。
関連：講座⑤ Stable Diffusion入門

PNG形式 (ピングけいしき)

【例え話：背景が透明なシール】画像を保存する方法（ファイル形式）の一つで、背景を透明にしたまま保存できるのが最大の特徴なんだ。LINEスタンプは、このPNG形式で作るのがルールだよ。
関連：講座⑥ AIでLINEスタンプ制作入門

SEO (エスイーオー)

【例え話：Googleへの"自己紹介"】「検索エンジン最適化」の略。Googleみたいな検索エンジンに、「私のサイトは、こんなことで困っている人の役に立てますよ」って、礼儀正しく自己紹介をして、検索結果の上の方に表示してもらうための技術のことだよ。
関連：講座㉑ AIホームページ作成入門

Suno (スノ)

【例え話：魔法のジュークボックス】あなたが紡いだ言葉（歌詞）とイメージを伝えるだけで、オリジナルソングを生み出してくれる音楽生成AIだよ。
関連：講座⑬ 音楽生成AI Suno入門

VOICEVOX (ボイスボックス)

【例え話：AI声優事務所】個性豊かなAI声優さんたちに、あなたの書いたセリフを感情豊かに読み上げてもらえる、無料の音声合成ソフトだよ。
関連：講座⑮ VOICEVOXマスター編

YMM4 (ゆっくりムービーメーカー4)

【例え話：動画編集の厨房】あなたが作ったイラスト、声、音楽といった全ての素材を並べて、一本の「動画」という物語に仕上げるための、無料の動画編集ソフトだよ。
関連：講座⑯ YMM4マスター編
