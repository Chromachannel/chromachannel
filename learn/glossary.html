機械学習 (Machine Learning)

AIにたくさんの「お手本（データ）」を見せてあげて、AI自身にルールやパターンを学ばせること。「仕分けマスター・ピクト」で、きみがAIに「これはくだものだよ」って教えてあげるのが、まさにこの機械学習の体験だよ。

教師あり学習 / 教師なし学習

機械学習の、二つの異なる育て方だよ。「AIの赤ちゃん 育成アプリ」で体験したように、先生（あなた）が「これはりんごだよ」と正解を教えてあげるのが教師あり学習。反対に、AIに答えを教えず、見た目の特徴だけで自分なりに仲間分けさせるのが教師なし学習なんだ。

「チャット（おしゃべり）」をする「ロボット」のこと。AIと人間が、LINEみたいに自然な言葉で会話できるように作られたプログラムだよ。「AI博士の談話室」は、まさにこの技術で動いているんだ。

AIに「こんなことをしてほしい！」とお願いするための「指示文」のこと。まるでAIに魔法をかけるための「呪文」みたいなものだね。良い呪文を唱えられると、AIはもっとすごい力を発揮してくれるんだ。

プロンプトエンジニアリング

英数字

AI（人工知能）

人間みたいに、たくさんの情報から物事を考えたり、学んだりすることができる、賢いコンピュータプログラムのこと。まるで「見習いシェフ」のように、たくさんのレシピ（データ）を読んで、新しい料理（答え）を考えるのが得意だよ。

AI博士

このサイトの案内役をしてくれる、優しくて物知りなAIチャットボットのこと。あなたのどんな質問や悩みにも、共感を持って寄り添ってくれる、心強い味方だよ。

CSS

HTMLで作った骨格に、「デザイン」というお化粧をしてあげるための言葉だよ。壁紙の色を変えたり、家具を並べたりするイメージだね。「リアルタイムプレビューエディタ」の右側で書くのが、このCSSなんだ。

GPTs

ChatGPTなどを開発したOpenAI社が提供する、特定の目的に合わせてAIをカスタマイズできる機能のこと。このサイトの「AI博士」も、この技術を応用して作られているよ。

HTML

ウェブページの「骨格」を作るための言葉だよ。お家を建てる時の、柱や壁みたいなものだね。「リアルタイムプレビューエディタ」の左側で書くのが、このHTMLなんだ。

JavaScript

ウェブページに「動き」や「機能」という命を吹き込むための、プログラミング言語だよ。ボタンを押したら計算してくれる家計簿 や、絵文字が消えるパズルゲーム など、このサイトのほとんどのアプリが、このJavaScriptの魔法で動いているんだ。

localStorage

ウェブブラウザに備わっている、小さな「記憶装置」のことだよ。家計簿 やタスク管理ツール で入力したデータが、ページを閉じても消えずに残っているのは、このlocalStorageに、あなたのパソコンの中だけで、こっそり保存されているからなんだ。

Stable Diffusion

言葉（プロンプト）から、とてもリアルで美しいイラストを生成するのが得意なAIの名前。まるで「何でも描いてくれる魔法の絵筆」のようなツールだよ。

Suno

歌詞や曲のイメージを伝えると、世界に一つだけのオリジナルソングを作ってくれる音楽生成AIの名前。楽器が弾けなくても、誰でも作曲家になれるんだ。

VOICEVOX

入力したテキストを、ずんだもんなどの人気キャラクターの声で、感情豊かに読み上げてくれるAI音声合成ソフトの名前。商用・非商用問わず無料で使える、すごいツールなんだ。

YMM4 (ゆっくりムービーメーカー4)

VOICEVOXと連携して、キャラクターが喋る動画などを簡単に作れる、無料の動画編集ソフトの名前だよ。
